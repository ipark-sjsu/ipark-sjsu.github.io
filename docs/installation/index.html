    Installation Guide #

    To install this project, follow these steps:

    1. Install Hugo
       conda install -c conda-forge hugo
    1. Clone the Repository
    git clone https://github.com/<user>/<user>.github.io.git
    1. Run the Hugo Server
    hugo server -D
